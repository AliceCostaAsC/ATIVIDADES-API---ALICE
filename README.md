# API_Temperatura

#Como Funciona

O Projeto foi criado através da API https://openweathermap.org/api, ele busca a temperatura de todas as cidades do Brasil e as apresenta.

# Referência

https://openweathermap.org/api

# Tecnologias Usadas

Html5, Css3, JavaScript e Bootstrap
